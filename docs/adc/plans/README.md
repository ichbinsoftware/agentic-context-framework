# Execution Plans

This directory contains step-by-step execution plans. A plan may be linked to an ADC record, but can also exist independently for tasks that need sequencing without a formal decision record.

Each plan file follows this naming convention:

```
YYYY-MM-DD--short-feature-slug.plan.md
```

---

> **Agent:** Do not read files in this directory unless you are actively executing a specific task that references a plan by ID, or you have been explicitly instructed to consult one. Do not scan this directory to build context. See `docs/adc/README.md` for the full retrieval policy.
